# Difficulties

## Normal
Intended difficulty.
Quicksaves, checkpoints and level start saves.

## Hard
Some challenge here.
Checkpoints and level start saves.
2x monster HP.

## Extreme
For veterans only.
Level start saves.
2x monster HP, speed, damage.

## Nightmare
Don't try this at home.
Permadeath.
3x monster HP, speed, damage.
Monsters respawn.
2x pickups.
